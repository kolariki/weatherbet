# WeatherBet Frontend

Interfaz de usuario para la plataforma de predicción climática.

## Setup

```bash
cp .env.example .env
# Configura tus credenciales de Supabase
npm install
npm run dev
```

## Tech Stack

- React 18 + Vite
- Tailwind CSS (dark theme)
- Lucide React (iconos)
- React Router v6
- React Hot Toast
- Supabase Auth
